# CG - PhysGaussian Project



## 实现目标

使用 PhysGaussian 的框架。



##### 改进内部填充方法

以下操作全部在 `\particle_filling\filling.py` 文件中进行。

1. 编写 `DBSCAN_cluster` 函数，实现 FastPhysGS 提到的聚簇算法 (DBSCAN)。
2. 编写 `fill_particles_MCIS` 函数，实现 FastPhysGS 提到的填充方法 (MCIS 采样)。
3. 修改 `fill_particles` 以及其它必要函数，为原本的内部填充方法提供逐物体填充的支持。
4. (optional) 按照 VLM 的返回结果 (按下文所示)，为每个物体调用不同的填充方式。



##### VLM 识别

参考 FastPhysGS。可以选择以下两种实现手段中的任意一种：

###### 第一种（在线使用 VLM）

1. 在 `\utils\vlm_utils.py` 中编写 `generate_bounded_image` 函数，为 VLM 生成带 aabb 绑定框以及编号的图片。
2. 在 `\utils\vlm_utils.py` 中编写 `get_initial_params` 函数。其将预先设计好合适的提示词以及上述图片一并自动提供给 VLM，让他提供对于对应编号的物体的杨氏模量的预测值。用一个 list (第 i 个元素存储了第 i 个物体的杨氏模量) 作为这个函数的返回值即可。
3. (optional) 让 VLM 同时预测物体的空心程度。

###### 第二种（离线使用 VLM）

1. 在 `\utils\vlm_utils.py` 中编写 `generate_bounded_image` 函数，为 VLM 生成带 aabb 绑定框以及编号的图片。
2. 设计合适的提示词，与上述图片一起手动提供给 VLM，让他提供对于对应编号的物体的杨氏模量的预测值。
3. (optional) 让 VLM 同时预测物体的空心程度。
4. 手动将 AI 输出的数据保存在 `\generated_data\vlm_data` 文件夹下。
5. 在  `\utils\vlm_utils.py` 中编写 `get_initial_params` 函数，从上述文件中读取出一个 list 并返回即可 (list 内容同上)。



##### 实现对 AI 生成视频的解码

1. 设计合适的提示词，让 AI 根据带 aabb / 不带 aabb 的单视角图片 (具体用哪种完全无所谓，看个人喜好) 生成视频 (离线生成即可)。将视频放在 `\generated_data\video` 文件夹下。
2. (optional) 让 AI 自行判断并给出关键帧 (存在碰撞的帧等)。
3. 在 `\utils\video_utils.py` 文件种编写 `get_video_frames` 函数，从视频格式读出帧序列并返回。



##### 实现 PhysDreamer 的优化算法

1. 更改 `\gs_simulation.py` 文件等，实现逐物体编码材质。
2. 更改 `\mpm_solver_warp` 文件夹，以将 PhysDreamer 的可微物质点法移植过来，并将其更改为逐物体显式编码形式。

## 附录：论文总结

###### Triplane：https://blog.csdn.net/flow_specter/article/details/130677562

一种新型场表达，思想和 nerf 接近。其由一个特征处理功能和一个小型 MLP 构成，接受一个三维坐标作为输入，返回该坐标下的物理量。

Triplane 材料场和逐物体材料的优缺点分析：

1. Triplane 与逐物体材料均要求单个物体由同种材料制成。
2. Triplane 允许同一物体的杨氏模量不均匀，逐物体不行。
3. Triplane 是隐式表达可能会脱离现实；逐物体更加显式，虽然自由度不高但不容易脱离现实。

#### PhysDreamer

##### 对问题的形式化

1. 材料模型：只实现了 Fixed Corotated 弹性模型。允许每个粒子获得任意材料参数。密度通过计算得到。
2. 学习目标：杨氏模量和初始速度。
3. 损失评价：令渲染图像和 AI 图像的差异最小。
4. 目标表达：用 triplane + MLP 的方式表达速度场和材料参数场。
5. 其他：使用类似 MLS-MPM 的物质点法。

##### 新的训练方法

1. 先随机生成杨氏模量，随后用前三帧学习初始速度直至收敛。
2. 之后定死速度，逐帧学习杨氏模量。
3. 次采样：用 K-means 方法聚类出 “驱动粒子”，只对驱动粒子进行仿真；但是渲染时仍然需要使用全部的高斯球，因此可以用实际高斯球在原始空间中（变形空间无法追踪）离他最近的八个驱动粒子线性插值得到实际高斯球的位置和协方差。



###### DBSCAN：https://zhuanlan.zhihu.com/p/1925351611942809908

一种聚类方法。主要思想为先选取出核心点，然后用扩散的方法建立点群。

#### FastPhysGS

##### 对问题的形式化

1. 材料模型：内置了数种弹性模型。认为每一个物体都由相同材料构成（即所属粒子的材料模型和材料参数完全相同，密度均匀分布）。
2. 学习目标：杨氏模量。
3. 损失函数：自监督。令应力张量和变形梯度维持在合理范围内 (主目标是最小化应力张量，但这么做有可能导致物体直接变成刚体，故引入惩罚项（变形梯度与单位矩阵的距离）来做折中)。
4. 目标表达：逐物体地，显式地表达这些物理量。
5. 其他：为每一个物体分配一个标签，并按标签组织内存。将组成物体的原生粒子/填充粒子放到他的标签对应的内存中。同样使用类似 MLS-MPM 的物质点法，但使用了一种 “多组分复合” 的改进。

##### 新的内部填充方法

1. 用 DBSCAN 将点云按所属物体分类成簇 C_i。
2. 用 quickhull (https://zhuanlan.zhihu.com/p/166105080) 算法得出每个簇的凸包 (形式为三角网格) H_i。
3. 获取凸包的 aabb 绑定框 B_i，随后从中均匀的，随机的选取 M (应远远大于对应物体中原生高斯球的数量) 个点作为候选填充点 Q_i。
4. 用光线传播算法 (只考虑凸包内粒子贡献的不透明度场，随便选一个方向计算光线传播) 确定初步内点集合 Q1_i。
5. 用每一个点与 ”他所在的簇的边界“ 的距离作为他的初步权，然后用类似 softmax 的方式对一个簇中的所有点的权作归一化得到最终权。按照这个权进行采样以得到内部点集合。

他的问题在于本质上是一种 “非凸/条状物” 特化的内部填充。如果是一个大球，他的表现会非常非常差。正好相反，原本的填充方式是一种 “凸/块状物” 特化的内部填充。如果是一个竹筐，则表现会非常差。

或许可以让视觉模型也来接管这一部分工作，决定为每个物体使用原本的填充，不填充还是用这种填充。（这个视觉模型或许需要自己训练，或者需要找到一个足够具体，足够清晰的，对这个要求的描述，然后交给现成的通用视觉模型）

##### 新的训练方法

1. 用 VLM (千问3) 来逐物体地预测材料模型（或许可以先渲染一张带 aabb 绑定框的图片再输入给 AI）、初始材料参数以及初始密度。
2. 单纯的将这个世界逐物体的分为多个部分，将每一个部分在该处的应力相加即得那个位置的总应力张量。
3. 先跑一遍完整的 MPM，允许每一帧进行多次 p2g2p；只存储初始帧、终止帧和某一中间帧。
4. 接下来，“假装” 跑一遍 MPM，但不实际计算，只是根据仿真过程构建梯度传播路径。
5. 根据这个路径做 E 的反向传播。

最大的优势是自监督。一方面它定义了一个很有意思而且优化起来不难的损失函数，另一方面它不需要视频数据辅助。这使得它训练速度极快，同时占用内存也不大。

它这么做的问题应该还是与物理脱节。“符合物理实际” 和 “物理上合理” 是有差距的。可能可以将他作为辅助优化目标或者某种正则项，但是光用这个目标优化可能还是达不到仿真的目的。
