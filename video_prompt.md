# VLM Prompt

请观察这张带有编号绑定框的图片。每个编号框表示一个需要进行物理仿真的物体。

你的任务是为每个编号物体估计初始物理参数：

1. Young's modulus / 杨氏模量 E，单位 Pa。
2. 可选：推荐的粒子填充方法 filling_method。

请根据物体的视觉外观、常见材质和物理直觉估计。对于植物、软枝叶、盆栽、布料、软塑料等，应给出较低或中等的杨氏模量；对于陶瓷、金属、玻璃、石头等刚性物体，应给出较高的杨氏模量。

本项目可接受的 filling_method 只有以下三种：

- "legacy"：适合凸、块状、实体物体。
- "mcis"：适合细长、非凸、枝条状、空洞较多或结构复杂的物体。
- "none"：不需要额外内部填充。

请只输出严格 JSON，不要输出解释、Markdown 或额外文字。格式如下：

```json
{
  "E": [数值1, 数值2, ...],
  "filling_method": ["方法1", "方法2", ...]
}
```

要求：

- 数组下标必须对应图片中的编号。例如编号 0 的结果放在数组第 0 个位置。
- E 必须是数字，不要写单位字符串。
- filling_method 必须从 "legacy"、"mcis"、"none" 中选择。
- 如果只有一个编号物体，也必须输出数组。

针对当前 ficus 测试图，一个合理输出示例如下：

```json
{
  "E": [2000000],
  "filling_method": ["mcis"]
}
```


请根据这张图片生成一段短视频。图片中包含一个由多个编号框标注的沙发/坐垫/枕头场景。

视频目标：
模拟上方枕头和坐垫在重力作用下向下压到沙发支撑结构上的运动。整体应符合柔软物体和支撑结构的物理直觉。

运动要求：
- 编号 0、3、4 的上方枕头区域应表现为柔软枕头，轻微下落、压缩、弯折。
- 编号 1 的中间坐垫应较柔软，在受压时发生明显但不过度的形变。
- 编号 2 的下方沙发/篮状支撑结构应基本保持稳定，只允许极小形变。
- 物体之间应有接触、挤压和缓慢回弹，不要穿透。
- 运动幅度不要夸张，保持真实、稳定、物理可信。

画面要求：
- 保持单视角侧面观察。
- 不要改变物体身份、材质、颜色或编号框含义。
- 不要新增物体、人物、文字、水印或镜头切换。
- 输出 3 到 5 秒视频，流畅、稳定、无闪烁。
